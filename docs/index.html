OVERVIEW

두 가지 상반된 요소가 합쳐진 브랜드명을 가진 ‘젠틀몬스터’는 누구나 겉모습은 젠틀하지만, 패션에 대한 욕망을 표현하고 싶은 마음을 안에 품고 있고, 이것을 디자인에 반영하겠다는 레이블의 의지가 녹아있다. 젠틀몬스터는 모두를 놀라게 할 새로움을 가장 중요한 가치로 두고 비주얼 콜라보레이션과 스페이스 마케팅을 통해 브랜드의 가치를 전달한다.

AS IS

FullPage Video 위주로 구성된 메인페이지

메인페이지의 FullPage 영상에 GNB 요소가 상대적으로 가려지는 현상 발생

전체적인 단순구성 레이아웃으로 인한 획일적인 느낌

TO BE

브랜드의 정체성을 전달하는 핵심 요소인 스페이스 마케팅과 비주얼 콜라보를 소개하는 섹션을 추가한다.

GNB의 요소들을 강조하여 가시성을 확보하고 우측의 메뉴바를 통해 접근성을 높인다.

불필요한 페이지 구성은 줄이고 아트적인 이미지 배치와 동적 요소를 추가하여 사용자와의 공감각적 인터렉션을 형성한다.

MAINPAGE

header

브랜드와 관련된 주요 카테고리를 배치하고, 애니메이션 효과를 통해 가시성을 높인다.

video-section

최신 브랜드 이슈와 관련된 비디오 컨텐츠를 통해 시각적 흥미를 유도한다.

menu-bar

사용자의 서비스와 관련된 카테고리를 아이콘으로 간결하게 표현하고, 고정 사이드 바 형태로 노출시켜 사용자의 접근성을 높인다.

collaboration-section

실제 소비자가 콜라보 제품을 착용한 이미지를 배치하여 제품에 대한 간접적인 경험을 유도한다.

offline store-section

오프라인 매장의 이미지와 컨셉 등의 정보를 마우스 hover 효과와 접목하여 시각적 흥미를 유도한다.

footer

카테고리에 따른 항목을 그룹화하여 사용자의 접근성을 높인다.

SUBPAGE - SUNGLASS

사용자가 제품 이미지에만 집중할 수 있도록 3*3 배치로 공간을 과감하게 사용한다. 동시에 모델이 제품을 착용하고 있는 이미지나 비디오 컨텐츠를 제품 이미지 사이에 배치한다.

결과적으로 사용자가 제품을 실제로 착용해볼 수 없는 오프라인의 한계를 줄일 수 있도록 간접적인 경험을 제공한다.

SUBPAGE - GLASS

썸네일 이미지를 클릭하면 제품 이미지를 확인할 수 있도록 제작하여 사용자의 편의성을 높인다. 또한, 같은 컬렉션 라인의 제품이나 사용자의 관심사와 비슷한 제품을 추천하여 사용자의 구매를 유도한다.

SUBPAGE - COLLABORATION

반복적인 배치의 지루함을 없애기 위해 지그재그 형태의 레이아웃 구조를 활용한다. 콜라보레이션과 관련된 이미지 또는 비디오를 배경으로 배치하여 시각적인 흥미를 유도하고, 콜라보 제품을 확인할 수 있도록 대표 제품 이미지를 노출시킨다. 또한, 불필요한 페이지 이동을 줄이기 위해 필요한 정보만 함축하여 사용자가 필요한 정보를 빠르게 습득할 수 있도록 유도한다.

SUBPAGE - STYLE

스크롤 애니메이션과 클릭 애니메이션을 활용하여 사용자와의 인터렉션을 유도한다. 사용자가 한 페이지 안에서 이미지 속 착용 제품의 정보를 확인하고, 제품 상세 페이지로 이동할 수 있도록 고려하여 불필요한 페이지를 제거하고 사용자의 편의성을 높인다.

SUBPAGE - STORE

반복적인 배치의 지루함을 없애기 위해 지그재그 형태의 레이아웃 구조를 활용한다. 불필요한 페이지 스크롤을 줄이기 위해 스토어 카테고리에 따라 내용을 분류, 사용자에게 필요한 정보만 제공한다.
